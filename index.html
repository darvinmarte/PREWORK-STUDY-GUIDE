HTML

  • Enter your HTML notes here
  • The head element contains information about the webpage
  • The head element is a container for metadata
  • Meta elements contain information about the page that is used by the browser
  • The user wont see information in the head or meta elements.
  • Body element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element.
  • H1, h2, h3, h4, h5, and h6 elements represent the level of heading a given text block represents
  • P element represents a paragraph or block of text
  • Ul, ol, and li elements represent unordered lists, ordered lists, and list items
  • Img element contains information about images that are displayed on the webpage, src attribute is the most important because it defines the location of the image
  • `Alt` attribute contains a text string that describes the appearance and functionality of the image.
  • Br element creates a line of empty space, or a line break, between two blocks of content
  • A element (which stands for anchor) creates links to the same webpage or other webpages, href attribute points to the URL for the link
  • Footer appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.
